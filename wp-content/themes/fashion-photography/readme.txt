== Description ==

Fashion Photography is a stunning theme with perfect layout for any photography related website. The theme is compatible with many famous WordPress plugins like Contact form 7, WooCommerce etc. It is SEO-friendly that will help your website rank easily on major search engines like Google. It is best suitable for photo album, photo portfolio, photo blogging, photo sharing, web designer, baby photography, wedding photographer, photo shooting, newspaper, publishing, travel, food, sports, life style, bloggers, fashion blogging, wedding photoshoot, hospitality business, and other websites. The page load times is faster as it is optimized for speed. You can shortcodes and custom CSS/JS to it to improve the functionality of your website. It is translation-ready and supports RTL layout. == Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Fashion Photography in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
